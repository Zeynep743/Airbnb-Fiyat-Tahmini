# Airbnb-Fiyat-Tahmini
Makine öğrenmesi yöntemleri ile Airbnb ev fiyatlarının tahmin edilmesi projesi.
# İstanbul Airbnb Fiyat Tahmini ve Karar Destek Sistemi

# Proje Hakkında

Bu proje, İstanbul'daki Airbnb ilanlarının gecelik fiyatlarını makine öğrenmesi yöntemleri kullanarak tahmin etmek ve kullanıcıların konaklama kararlarını desteklemek amacıyla hazırlanmıştır.

## Özellikler

- Makine öğrenmesi ile gecelik fiyat tahmini
- Konum analizi
- Turistik noktalara uzaklık hesaplama
- Tahmini taksi maliyeti hesaplama
- Bütçe planlama
- Türkçe ve İngilizce dil desteği

## Kullanılan Teknolojiler

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Folium

## Proje Yapısı

- `app.py` : Streamlit kullanıcı arayüzü
- `main.py` : Veri analizi, özellik mühendisliği ve model geliştirme
